# Students-Performance-in-Exams
Este projeto aplica aprendizado supervisionado ao Student Performance Data Set, que contém informações sobre alunos do ensino secundário em Portugal, com o objetivo de comparar Redes Neurais Artificiais e Regressão Simbólica na predição do desempenho acadêmico, analisando tanto a acurácia dos modelos quanto sua capacidade de interpretação.

[Link database](https://archive.ics.uci.edu/dataset/320/student+performance)
